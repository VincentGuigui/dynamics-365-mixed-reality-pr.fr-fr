# <a name="dynamics-365-remote-assist-faq"></a>FAQ Dynamics 365 Remote Assist

Voici les réponses aux questions fréquentes sur l'utilisation de[!include[cc-microsoft](../includes/cc-microsoft.md)] [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)].

Pour plus d'informations, voir [Obtenir de l'aide sur HoloLens](https://support.microsoft.com/products/hololens).

Si vous êtes un administrateur informatique et que vous souhaitez déployer [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)] sur des appareils exécutant [!include[pn-hololens](../includes/pn-hololens.md)] Commercial Suite, voir [Acheter et déployer Dynamics 365 Remote Assist](buy-and-deploy-remote-assist.md). 

## <a name="can-i-try-the-app-before-buying-it"></a>Puis-je essayer l'application avant de l'acheter ?

Oui, [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)] est disponible en essai gratuit de 90 jours. Pour plus d'informations, voir [Essayer Dynamics 365 Remote Assist gratuitement](try-remote-assist-free.md). Veuillez noter que l'essai gratuit de 90 jours est disponible uniquement pour les utilisateurs qui n'ont pas encore acheté un abonnement. 

## <a name="what-are-the-technical-requirements-for-installing-dynamics-365-remote-assist"></a>Quelles sont les exigences techniques pour installer Dynamics 365 Remote Assist ?

Pour utiliser [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)], vous aurez besoin des éléments suivants :

-   [Un abonnement à Dynamics 365 Remote Assist.](../licensing/buy-and-deploy.md)

-   Un [HoloLens exécutant la Mise à jour d'avril 2018 de Windows 10](https://support.microsoft.com/help/12643). 

-   Pour effectuer un appel vidéo, vous avez besoin d'un contact utilisant la dernière version de [Microsoft Teams](https://products.office.com/microsoft-teams/group-chat-software) sur un PC exécutant Windows 10 ou un appareil mobile exécutant [!include[pn-microsoft-teams](../includes/pn-microsoft-teams.md)] Mobile. [!include[pn-microsoft-teams](../includes/pn-microsoft-teams.md)] est [disponible en téléchargement gratuit](https://teams.microsoft.com/downloads).

## <a name="do-i-need-a-subscription-to-microsoft-teams-or-office-365"></a>Ai-je j'ai besoin d'un abonnement à Microsoft Teams ou Office 365 ?

Un abonnement à Teams est inclus dans l'abonnement [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)] pour les personnes avec une licence [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)] sur [!include[pn-hololens](../includes/pn-hololens.md)]. Une licence [!include[pn-microsoft-teams](../includes/pn-microsoft-teams.md)] est également requise pour les personnes (spécialistes) qui communiquent avec un utilisateur de [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)] sur [!include[pn-hololens](../includes/pn-hololens.md)]. [!include[pn-teams](../includes/pn-teams.md)] peut être disponible [en téléchargement gratuit](https://teams.microsoft.com/downloads) pour ces utilisateurs. 

## <a name="how-often-is-dynamics-365-remote-assist-updated"></a>À quelle fréquence l'application Dynamics 365 Remote Assist est-elle mise à jour ?

[!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)] suit la Politique moderne en matière de cycle de vie avec des mises à jour continues de service et des mises à jour importantes tous les 6 mois (octobre/avril). 

## <a name="how-can-i-tell-what-version-im-using"></a>Comment indiquer la version que j'utilise ?

Allez à **Paramètres**, puis trouvez le numéro de version dans la section **À propos de**.

## <a name="what-languages-is-dynamics-365-remote-assist-available-in"></a>Dans quelles langues Dynamics 365 Remote Assist est-elle disponible ?

Pour le moment, [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)] est disponible en anglais et japonais.

## <a name="what-languages-are-voice-commands-and-speech-recognition-available-for"></a>Pour quelles langues les commandes vocales et la reconnaissance vocale sont-elles disponibles ?

Actuellement, pour HoloLens 1, les commandes vocales et la reconnaissance vocale sont disponibles uniquement en anglais. Pour HoloLens 2, les commandes vocales et la reconnaissance vocale sont disponibles pour les langues suivantes :

-   Anglais (États-Unis)
-   Anglais (Canada)
-   Anglais (Royaume-Uni)
-   Anglais (Australie)
-   Chinois (Chine)
-   Français (France)
-   Français (Canada)
-   Allemand (Allemagne)
-   Italien (Italie)
-   Japonais (Japon)
-   Espagnol (Espagne)

## <a name="where-is-dynamics-365-remote-assist-available"></a>Où Dynamics 365 Remote Assist est-elle disponible ?

[!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)], qui nécessite [!include[cc-microsoft](../includes/cc-microsoft.md)] [!include[pn-hololens](../includes/pn-hololens.md)], est disponible dans le monde entier.

Les appareils [!include[pn-hololens](../includes/pn-hololens.md)] sont disponibles sur les marchés suivants (en anglais uniquement) :

Autriche, Australie, Belgique, Bulgarie, Canada, Chine, Croatie, Chypre, République tchèque, Danemark, Estonie, Finlande, France, Allemagne, Grèce, Hongrie, Islande, Irlande, Italie, Japon, Lettonie, Liechtenstein, Lituanie, Luxembourg, Malte, Pays-Bas, Nouvelle-Zélande, Norvège, Pologne, Portugal, Roumanie, Slovaquie, Slovénie, Espagne, Suède, Suisse, Turquie, Royaume-Uni, États-Unis

## <a name="why-am-i-not-getting-incoming-call-notifications"></a>Pourquoi ne reçois-je pas de notifications d'appels entrants ?

Cela peut se produire si vous êtes connecté à [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)] avec un compte différent de celui que vous avez utilisé pour vous connecter à votre [!include[pn-hololens](../includes/pn-hololens.md)]. Assurez-vous que vous êtes connecté tant à l'application et qu'à l'appareil avec le même compte. 

## <a name="i-see-a-message-saying-i-have-no-internet-connection"></a>Je vois un message indiquant que je n'ai aucune connexion Internet.

Pour utiliser [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)], vous devez disposer d'une connexion Wi-Fi. Assurez-vous que votre [!include[pn-hololens](../includes/pn-hololens.md)] est [connecté au Wi-Fi](https://support.microsoft.com/help/12629).

## <a name="dynamics-365-remote-assist-is-frozen-or-unresponsive"></a>Dynamics 365 Remote Assist est bloquée ou ne répond pas.

Essayez de fermer et de rouvrir [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)]. Si vous rencontrez encore des problèmes, [redémarrez votre HoloLens](https://support.microsoft.com/help/12642).

## <a name="dynamics-365-remote-assist-isnt-showing-my-contacts"></a>Dynamics 365 Remote Assist n'affiche pas mes contacts.

Si vous ne voyez pas les contacts prévus, essayez ce qui suit :

-   Vérifiez que vous vous êtes connecté à [!include[cc-microsoft](../includes/cc-microsoft.md)] [!include[pn-teams](../includes/pn-teams.md)] au moins une fois avec le compte que vous utilisez sur votre [!include[pn-hololens](../includes/pn-hololens.md)].

-   Fermez et rouvrez [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)].

-   [Redémarrez votre HoloLens](https://support.microsoft.com/help/12642).

## <a name="the-mixed-reality-toolbar-doesnt-appear-on-the-pc-teams-client-during-a-dynamics-365-remote-assist-call"></a>La barre d'outils de réalité mixte ne s'affiche pas sur le client Teams sur PC lors d'un appel Dynamics 365 Remote Assist.

Cela peut se produire si [!include[pn-teams](../includes/pn-teams.md)] a téléchargé une nouvelle version du client en arrière-plan mais exécute toujours la version plus ancienne de l'application. Le redémarrage de votre PC résoudra généralement ce problème.

## <a name="why-does-audio-sometimes-cut-out-even-in-quiet-environments"></a>Pourquoi l'audio coupe-t-il parfois même dans les environnements tranquilles ?

Si [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)] est utilisée dans un environnement bruyant, utilisez les casques aux deux extrémités pour améliorer la communication. L'utilisation de casques pour les deux utilisateurs empêche toute filtration d'écho d'activer proactivement la suppression du bruit, ce qui coupe l'audio.

## <a name="can-i-use-dynamics-365-remote-assist-to-place-a-call-between-two-separate-tenants"></a>Puis-je utiliser Dynamics 365 Remote Assist pour passer un appel entre deux tenants distincts ?

Oui, mais uniquement si l'expert appelle en tant qu'invité sur le tenant de l'autre utilisateur.
Actuellement, [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)] ne prend pas en charge deux utilisateurs s'appelant, même si deux tenants sont joints par fédération.

## <a name="does-microsoft-teams-include-the-same-dynamics-365-remote-assist-features-on-all-platforms-and-devices"></a>Microsoft Teams comprend-elle les mêmes fonctions Dynamics 365 Remote Assist sur toutes les plateformes et tous les appareils ?

Non. L'encre, les flèches et l'ajout de fichiers ne sont disponibles que sur [!include[pn-teams](../includes/pn-teams.md)] pour PC.

## <a name="i-signed-in-to-my-hololens-but-i-still-cant-sign-in-to-dynamics-365-remote-assist"></a>Je me suis connecté à mon HoloLens, mais je ne peux toujours pas me connecter à Dynamics 365 Remote Assist.

Si vous rencontrez des problèmes pour vous connecter à [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)], essayez ce qui suit :

-   Assurez-vous que vous utilisez un compte professionnel ou scolaire, et non pas un compte [!include[cc-microsoft](../includes/cc-microsoft.md)], pour vous connecter à [!include[pn-hololens](../includes/pn-hololens.md)].

-   Assurez-vous que le compte professionnel ou scolaire que vous utilisez a un abonnement valide à [Office 365 Business](https://products.office.com/business/office) Premium ou Essentials avec [Microsoft Teams](https://products.office.com/microsoft-teams/group-chat-software) activé.

-   Assurez-vous que votre [!include[pn-hololens](../includes/pn-hololens.md)] est connecté à Internet.

-   Vérifiez avec votre administrateur pour vous assurer que votre appareil et votre réseau sont configurés pour fonctionner avec [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)].

## <a name="how-do-i-set-up-dynamics-365-field-service-to-work-with-dynamics-365-remote-assist"></a>Comment configurer Dynamics 365 Field Service pour fonctionner avec Dynamics 365 Remote Assist ?

Pour savoir comment configurer l'intégration de [!include[pn-dyn-365-field-service](../includes/pn-dyn-365-field-service.md)] avec [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)] afin qu'un collaborateur de première ligne puisse appeler un expert dans le contexte d'un ordre de travail [!include[pn-dyn-365-field-service](../includes/pn-dyn-365-field-service.md)], voir [Intégrer Dynamics 365 Field Service avec Dynamics 365 Remote Assist](troubleshoot-field-service.md).

## <a name="is-group-calling-available-in-dynamics-365-remote-assist"></a>L'appel de groupe est-il disponible dans Dynamics 365 Remote Assist ?

Oui, l'appel de groupe est disponible à compter de la version d'avril de [!include[pn-dyn-365-remote-assist](../includes/pn-dyn-365-remote-assist.md)]. Jusqu'à 50 personnes peuvent participer au même appel vidéo.

## <a name="what-file-formats-does-the-insert-file-feature-support"></a>Quels formats la fonctionnalité Insérer un fichier prend-elle en charge ?

La fonctionnalité Insérer un fichier prend en charge la plupart des formats d'image (JPG, GIF, PNG) et les fichiers PDF.

## <a name="can-i-share-files-from-sharepoint-on-dynamics-365-remote-assist"></a>Puis-je partager des fichiers depuis SharePoint sur Dynamics 365 Remote Assist ?

Non, vous pouvez uniquement partager des fichiers locaux et des fichiers issus de [!include[pn-onedrive](../includes/pn-onedrive.md)].

Besoin d'aide complémentaire ? Essayez nos [forums des utilisateurs](https://community.dynamics.com/365/remoteassist).
